# FinFlow Config Repo

This folder is a local stand-in for the GitHub config repository.

Each file name must match `spring.application.name` from the target service:

- `AUTH-SERVICE.properties`
- `APPLICATION-SERVICE.properties`
- `DOCUMENT-SERVICE.properties`
- `ADMIN-SERVICE.properties`
- `api-gateway.properties`
- `service-registry.properties`

When you move this to GitHub, keep the same file names.
